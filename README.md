# News App
News App is a native Android application for reading the news, which uses News.io API to fetch news using the Retrofit library. The application uses MVVM architecture and written in Kotlin.

* **NewsData APIs**: [_Documentation_](https://newsdata.io/documentation)

## Documentation

- **Language**: Kotlin
- **Architecture**: Model View ViewModel (MVVM)
- **Libraries**: [Retrofit](https://github.com/square/retrofit), [Shimmer Effect](https://github.com/facebookarchive/shimmer-android?tab=readme-ov-file), LiveData, ViewModel, Navigation Components.
